About the Game

How to Play videogame_asset Welcome to NBA Higher or Lower! The rules are simple: given two players who were in the top ten for a given stat category (points per game, rebounds per game, etc.), select the player you think had the higher stat using the "left" or "right" buttons. If you're correct, you get a point. If you're wrong, you have to start over. See how high of a streak you can get, and challenge your friends!
